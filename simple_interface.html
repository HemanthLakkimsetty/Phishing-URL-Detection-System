🚀 How to Use the Full Version

  1. Install Python: Make sure you have Python 3.7 or higher installed
  2. Install Dependencies: Run pip install -r requirements.txt
  3. Start Web Server: Run python web_interface.py or double-click start_web.bat
  4. Open Browser: Go to http://localhost:5000
  5. Paste URL: Enter any URL in the input box and click "Check URL Safety"
  6. View Results: See detailed safety analysis with color-coded results

💡 The full version performs real-time analysis of URLs including DNS checks, domain age verification, SSL certificate validation, and content analysis!
